## 文件目录

### 设定要求

* 不使用复杂的数据结构、尽可能使用链表、数组等简单数据结构
* 从指针开始维护、注重空间的分配与**回收**
* 尽可能给予真实bash的体验



### 目前已实现的功能

- 简单的语法分析
  - 自左向右按空格分割
  - 多空格并不会影响分割



* 实现shell一部分经典指令

  * `echo` 复读

  * `clear` 清屏

  * ` mkdir` 创造文件夹

  * `touch` 如果不存在则创造文件

  * `ls` 目录下文件

  * `>` 输出重定向

  * `>>` 输出重定向，追加模式

  * `exit` 退出

  * `rm`  删除文件

  * `rm -r` 递归删除文件夹

  * **支持绝对路径进行访问**
